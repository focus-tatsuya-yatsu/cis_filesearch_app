## 本文テンプレート

---

お疲れ様です。

CIS File Search に新機能「NASファイル直接オープン」を追加しましたのでご案内します。

### 🎯 機能概要

検索結果から、NAS上のファイルをワンクリックで直接開けるようになりました。

### 📥 ご利用方法

**初回のみ、簡単なセットアップが必要です。**

1. 下記リンクからインストーラーをダウンロード
   - [インストーラーのダウンロード]（リンクをここに挿入）

2. ダウンロードしたZIPファイルを展開

3. 「インストール.bat」をダブルクリック

※インストールは1分程度で完了します
※管理者権限は不要です

### 🖱️ 使い方

1. CIS File Search で検索を実行
2. 検索結果の **緑色のフォルダアイコン** をクリック
3. ファイルがExplorerで開きます

### ⚠️ 注意事項

- **Windows専用**です（Macをご利用の方は「パスをコピー」機能をご利用ください）
- **社内ネットワーク** または **VPN接続** が必要です
- 在宅勤務でVPN未接続の場合は、「ダウンロード」または「プレビュー」機能をご利用ください

### ❓ よくある質問

**Q: インストールしたのに動かない**

A: ブラウザを一度閉じて再起動してください。
   それでも動かない場合はIT管理者にお問い合わせください。

**Q: アンインストールしたい**

A: 展開したフォルダ内の「アンインストール.bat」をダブルクリックしてください。

ご不明な点がございましたら、IT部門までお問い合わせください。
