# CIS ファイル検索システム タスク管理

## 📊 進捗ダッシュボード

このディレクトリでは、プロジェクトの全タスクをフェーズごとに管理します。

### 📁 ファイル構成

```
docs/tasks/
├── README.md                    # このファイル
├── progress-dashboard.md        # 全体進捗ダッシュボード
├── phase1-tasks.md             # Phase 1: 基本機能実装
├── phase2-tasks.md             # Phase 2: 高度な機能実装
├── phase3-tasks.md             # Phase 3: AI機能実装
└── phase4-tasks.md             # Phase 4: 運用最適化
```

### 🎯 使い方

1. **進捗確認**: `progress-dashboard.md` で全体の進捗を確認
2. **タスク確認**: 各フェーズのファイルで詳細タスクを確認
3. **更新方法**: チェックボックス `[ ]` → `[x]` に変更してコミット

### ✅ タスクステータス

| 記号 | 意味 |
|------|------|
| `[ ]` | 未着手 |
| `[~]` | 進行中 |
| `[x]` | 完了 |
| `[-]` | 保留/スキップ |

### 📌 優先度

| 記号 | 意味 |
|------|------|
| 🔴 | 高優先度（クリティカルパス） |
| 🟡 | 中優先度 |
| 🟢 | 低優先度 |

### 👥 担当者略称

| 略称 | 役割 |
|------|------|
| PM | プロジェクトマネージャー |
| FE | フロントエンドエンジニア |
| BE | バックエンドエンジニア |
| QA | QAエンジニア |
| DevOps | DevOpsエンジニア |
| ALL | 全員 |
