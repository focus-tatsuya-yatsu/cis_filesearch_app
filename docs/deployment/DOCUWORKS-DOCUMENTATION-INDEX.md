## 🔍 よくある質問 (FAQ)

### Q1: どのドキュメントから読めばいい?
**A**: 役割によって異なります:
- **PM/意思決定者**: エグゼクティブサマリー
- **開発者 (急ぎ)**: クイックスタートガイド
- **開発者 (初めて)**: 詳細計画書

### Q2: 全部読む必要がある?
**A**: いいえ、必要な部分だけでOK:
- 全体把握: エグゼクティブサマリー (5分)
- 実行: クイックスタートガイド (必要なセクションのみ)
- 詳細理解: 詳細計画書 (必要なタスクのみ)

### Q3: コードはどこにある?
**A**:
- コマンド例: クイックスタートガイド
- 完全実装: 詳細計画書
- C#/.NETコード: 詳細計画書に全掲載

### Q4: トラブルが発生したら?
**A**:
1. クイックスタートガイドのトラブルシューティング
2. 詳細計画書の該当タスク確認
3. サポート連絡 (緊急時)

### Q5: Day 1だけ実施して、Day 2は後日でもいい?
**A**: はい、可能です。各Dayは独立しています。
ただし、ライセンス到着前にDay 2完了推奨。
